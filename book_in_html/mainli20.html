usando passos h = 0,1 e h = 0,01.

E 8.2.2. Use o Método de Euler e o Método de Euler melhorado para obter aproximações numéricas para a solução do seguinte problema de valor inicial para t ∈ [0,1]:

usando passo h = 0,1. Compare os valores da solução exata dada por y(t) = com os numéricos nos pontos t = 0, t = 0.1, t = 0.2, t = 0.3, t = 0.4, t = 0.5, t = 0.6, t = 0.7, t = 0.8, t = 0.9, t = 1.0.

Resposta.

No Scilab, esta tabela pode ser produzida com o código:

       deff(’dy=f(y,t)’,’dy=-y-y^2’)  
       sol_Euler=Euler(f,0,1,10,1)’  
       sol_Euler_mod=Euler_mod(f,0,1,10,1)’  
       deff(’y=y_exata(t)’,’y=1/(2*exp(t)-1)’)  
       t=[0:.1:1]’  
       sol_exata=feval(t,y_exata)  
       tabela=[t sol_exata sol_Euler sol_Euler_mod abs(sol_exata-sol_Euler) abs(sol_exata-sol_Euler_mod)]  
      

♢
